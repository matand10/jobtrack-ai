Project: JobTrack AI
Build a focused portfolio MVP for job seekers to track job applications and generate AI resume/cover-letter suggestions.
Primary goal: clean, deployed full-stack MVP, not huge startup app.
Stack:

- Frontend: Vite, React, TypeScript, Tailwind, React Router, React Query, React Hook Form, Zod
- Backend: Node.js, Express, TypeScript, PostgreSQL, Prisma, JWT, bcrypt, Zod
- AI: backend-only provider integration
- Deployment: Vercel frontend, Render/Railway backend, Neon PostgreSQL
  MVP features:
- Auth
- Applications CRUD
- Search/filter/sort
- Dashboard stats
- AI tailoring suggestions
